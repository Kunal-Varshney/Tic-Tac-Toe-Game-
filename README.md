# ❌⭕ Tic Tac Toe Game

A fun and interactive **Tic Tac Toe Game** built with **HTML, CSS, and JavaScript**.
Play against another player and enjoy a clean, responsive, and engaging UI.

---

## 🚀 Features

* Classic 2-player Tic Tac Toe gameplay
* Interactive and responsive game board
* Win detection logic (rows, columns, diagonals)
* Restart game functionality
* Clean and modern UI design

---

## 🛠️ Tech Stack

* **HTML** → Structure of the game
* **CSS** → Styling and layout
* **JavaScript** → Game logic and interactions

---

## 🎯 How to Play

1. Open the game in your browser
2. Player 1 starts with **X**, Player 2 uses **O**
3. Players take turns clicking on the grid
4. First player to align 3 marks (row, column, diagonal) wins
5. If all cells are filled → it's a draw
6. Click **Restart** to play again
